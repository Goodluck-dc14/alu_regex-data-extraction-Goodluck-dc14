# Regex Data Extraction - JavaScript

## 🚀 Project Overview
This project extracts specific data from text using **Regular Expressions (Regex)** in JavaScript.  
It processes large amounts of text to extract:
- ✅ Email Addresses  
- ✅ URLs  
- ✅ Phone Numbers  
- ✅ Currency Amounts
